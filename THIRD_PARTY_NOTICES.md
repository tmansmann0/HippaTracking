# Third-Party Notices

## rrweb

HippaTracking uses rrweb for optional session recording.

- Package: `rrweb`
- License: MIT
- Repository: <https://github.com/rrweb-io/rrweb>

The relay serves `rrweb/dist/record/rrweb-record.min.js` only when the hosted
runtime configuration enables session recording and the browser has granted
consent.
